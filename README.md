# github.com/yenbekbay/dotfiles

Ayan Yenbekbay's dotfiles, managed with [`chezmoi`](https://github.com/twpayne/chezmoi).

Install them with:

    bash -c "bash <(curl -fsSL https://raw.githubusercontent.com/yenbekbay/dotfiles/master/install.sh)"
